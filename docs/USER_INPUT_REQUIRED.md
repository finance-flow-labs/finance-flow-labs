# User Input Required (to finalize Ground Rules)

아래 항목이 확정되면 자동화 품질이 크게 올라갑니다.

1. **투자 우주 범위**
   - 미국 주식/ETF만? (KR/Global 포함 여부)
   - 암호화폐 포함 범위(BTC/ETH만 vs 알트 허용)

2. **리스크 한도**
   - 최대 허용 MDD(예: -15%, -20%, -30%)
   - 레버리지 자산 총합 상한(예: 10%, 20%, 30%)

3. **성과 기준**
   - 목표 수익률: 연 20%는 유지?
   - 비교 벤치마크: QQQ, S&P500, 60/40 중 무엇?

4. **평가 호라이즌**
   - 시그널 평가 주기: 1주/1개월/3개월 중 기본값

5. **집행 정책**
   - 모의투자만 자동, 실매매는 수동 승인 유지?
   - 리밸런싱 주기(주간/월간/분기)

6. **리포트 선호**
   - 텔레그램 보고 빈도(실시간/하루 1회/요청 시)
   - 보고 형식(짧은 3줄/상세)
